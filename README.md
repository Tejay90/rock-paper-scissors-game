## Rock, Paper, Scissors Game

An interactive Rock, Paper, Scissors game developed with JavaScript, HTML5, and CSS3. This project demonstrates the creation of a responsive web application featuring dynamic game logic, user input processing, and visual feedback.

## Features

- Choose to win in 3, 5, or 10 rounds via a dropdown menu.
- Clear color coding for wins (green), losses (red), and ties (yellow).
- Clean layout using CSS gradients and responsive layout principles.
- Real-time score and result updates without page reloading (SPA principle).
- Rules and instructions clearly structured directly within the interface.

## Technologies

- HTML5: Semantic structure and accessibility.
- CSS3: Variables (`:root`), Flexbox layout, gradients, and responsiveness.
- JavaScript: Event handling, DOM manipulation, and a random number generator for AI logic.
- Environment: Visual Studio Code.

## Installation & Setup

To run the game locally:

1. Clone the repository:
   git clone https://github.com/Tejay90/rock-paper-scissors-game.git

2. Open files: Open the index.html file directly in a modern web browser. No server is required as it is a pure client-side application.


## How to Play

1. Click the Rock, Paper, or Scissors button.
2. The system selects a random option.
3. The score is updated and the result is displayed immediately.
4. Select the target number of rounds to win at the top (3, 5, or 10).
5. After winning, a "Play again?" button appears.

## Rock, Paper, Scissors Game

Ein interaktives Stein-Schere-Papier-Spiel, entwickelt mit JavaScript, HTML5 und CSS3. Das Projekt demonstriert die Erstellung einer responsiven Webanwendung mit dynamischer Spiellogik, Benutzereingabe-Verarbeitung und visuellem Feedback.

## Funktionen

- Wählbare Siegbedingung (3, 5 oder 10 Runden) über ein Dropdown-Menü.
- Visuelle Unterscheidung zwischen Sieg (grün), Niederlage (rot) und Unentschieden (gelb).
- Sauberes Design mit CSS-Gradienten und modernen Layout-Prinzipien.
- Direkte Aktualisierung des Spielstands und der Ergebnisse ohne Neuladen der Seite.
- Klare Strukturierung der Regeln und Spielanleitung direkt in der Benutzeroberfläche.

## Technologien

- HTML5: Semantische Struktur und Barrierefreiheit.
- CSS3: Variablen (`:root`), Flexbox-Layout, Gradienten und Responsivität.
- JavaScript: Event-Handling, DOM-Manipulation, Zufallsgenerator für die KI.
- Entwicklungsumgebung: Visual Studio Code.


## Installation & Setup

Um das Spiel lokal auszuführen:

1. Repository klonen:
   git clone https://github.com/Tejay90/rock-paper-scissors-game.git
   
2. Dateien öffnen: Öffnen Sie die index.html direkt in einem modernen Webbrowser (kein Server notwendig, da reines Client-Side-Scripting).


## Spielanleitung

1. Klicken Sie auf die Schaltfläche Rock, Paper oder Scissors.
2. Das System wählt eine zufällige Option.
3. Der Spielstand wird aktualisiert und das Ergebnis angezeigt.
4. Wählen Sie oben die Anzahl der Runden für den Sieg (3, 5 oder 10).
5. Nach dem Spielgewinn erscheint ein "Neues Spiel"-Button.
